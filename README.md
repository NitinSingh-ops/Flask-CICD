# Flask CI/CD Pipeline

## Project Overview

This project demonstrates Continuous Integration and Continuous Deployment (CI/CD) for a Flask application using Jenkins and GitHub Actions.

## Technologies Used

- Python 3.13
- Flask
- Pytest
- Jenkins
- GitHub Actions
- Git

## Jenkins Pipeline Stages

1. Checkout
2. Build
3. Test
4. Deploy

## GitHub Actions Workflow

- Install Dependencies
- Run Tests
- Build Application
- Deploy to Staging
- Deploy to Production
